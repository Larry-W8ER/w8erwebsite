Larry W 8 ER





Hi! I'm Larry W8ER. I am located in Sandusky Ohio. Sandusky is on Lake Erie about half way between Cleveland and Toledo.'


My main radio is an Anan 7000DLE Mk II, the blackface. I am using an Intel NUC8i5 running Windows 10 to host Thetis, the program that runs the radio. On my desktop I use another Intel NUC8i5, which runs Linux Mint, to access the radio computer with desktop remote software. The desktop also uses remote desktop to acess the NUC7i5 in the basement, with the amplifiers, that provides the webcam allowing me to watch the amplifier while in operation.'


All of my antennas are half wave dipoles, centered on a 65 foot Rohn 25 tower in my back yard. There is a remote coax switch on the tower that allows me to select the antenna. The tower has (2) 7/8 inch hardline feeds to the house. One carries the ham radio duties and the other connects to the TV antenna at the top.
